## Currency Converter

This repository was created using [Nest](https://github.com/nestjs/nest).

## Project setup

```bash
$ npm install
```

Create a `.env` file with your currency API key, ie:

```
CURRENCY_API_KEY=key_goes_here
```

## Compile and run the project

```bash
$ npm run start
```

## Run tests

```bash
$ npm run test
```
